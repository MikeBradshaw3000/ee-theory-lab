Kit location and delivery
-------------------------

The kit lives at workspace root for this commit (typical session-handoff
delivery pattern). The convention for kit storage and revisability is
specified in protocols/foundational/canonical_artifacts_index.md Section
10: the kit is "typically delivered as a session-handoff artifact under
claude_session_handoffs/YYYY-MM-DD[-N]/ rather than committed to the
repository tree." This commit places it at workspace root for
versioned-history capture; session-handoff folders contain copies for
delivery.

Whether to standardize on a stable repository path for the kit (e.g.,
protocols/kit/) versus continuing the handoff-folder pattern is a future
question; it is registered here for future kit-revision work but does
not block kit-revision-3.
